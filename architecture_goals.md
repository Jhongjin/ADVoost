\# ADVoost Clone Project Goals

\- Tech Stack: Next.js (App Router), Python (FastAPI), SQLite

\- Core Logic: 100점 만점 시작 감점 방식 (A\~F 등급)

&#x20; - FAIL (-15점): URL 접속 실패, robots.txt 거부, 스크립트 에러

&#x20; - WARNING (-5점): title 누락/중복, OG 누락, 모바일 뷰포트 미흡

&#x20; - WARNING (-2점): img alt 누락, H1 다중 발견

\- Grade Cut: A(>=90, FAIL 0), B(>=80, FAIL 0), C(>=65 또는 FAIL <=3), D(>=50), F(<50 또는 수집 FAIL)

\- Exception: 동일 URL 72시간 이내 재진단 시 크레딧 차감 없이 리프레시
